# GitHub Profile

本仓只持有 `gaofeng21cn` GitHub profile README 与随附展示资产。

- `README.md` 是公开入口；项目状态、安装说明和 currentness 必须引用对应 owner 仓库，不在这里复制第二份实现或发布 truth。
- 保持个人资料、链接、品牌名称和公开资产一致；不要提交私有联系方式、凭据、机器信息或未公开项目状态。
- 修改后运行 `git diff --check`，并检查 Markdown/HTML 链接、图片路径和 GitHub profile 渲染可读性。

## CodeGraph

- 本仓库使用本地 `.codegraph/` 索引；该目录不得纳入 Git。
- 定义、调用、影响范围和代码路径等结构检索优先使用 CodeGraph；字面文本检索使用 `rg`。
- 索引缺失或过期时运行 `codegraph init .` 或 `codegraph sync .`。
